Getting Started

Prerequisites: A Claude subscription (Pro, Max, or Teams), Git, and your preferred statistical software (R, Python, or Stata).

Install Claude Code

macOS / Linux

npm install -g @anthropic-ai/claude-code

Requires Node.js 18+. Verify with claude --version.

Windows

Open PowerShell and run:

irm https://claude.ai/install.ps1 | iex

Requires Git for Windows. See the full Windows installation guide for detailed steps and troubleshooting.

Quick Start

  1. Install Claude Code (see above)
  2. Download and unzip the tutorial package
  3. Open a terminal and cd into the tutorial/ folder
  4. Run claude to start a session
  5. Type /tutorial start to begin the interactive walkthrough

The Tutorial

A 20-minute interactive walkthrough that teaches you to use Claude Code for a complete data analysis workflow. You’ll work with a simulated dataset on union membership and wages in Midwest states (2010–2019), moving from messy raw data to clean analysis: diagnosing data quality issues, cleaning variables, generating an exploration report with tables and figures, and thinking about causal identification strategies.

The tutorial runs inside Claude Code itself—just type /tutorial start after opening a session in the tutorial folder. It works with R, Python, or Stata. Reference solutions in all three languages are included.
